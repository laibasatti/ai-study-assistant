🎓 AI Study Assistant

An AI-powered study assistant built for university students. Ask any question and get instant smart answers!

 ✨ Features
- 💬 Real-time AI answers
- 🎨 Modern Gen Z design
- 💾 Chat history
- 🌙 Beautiful dark theme
- 🚀 Instant responses

🛠️ Built With
- Python
- Streamlit
- Groq API
- Llama 3.3 Model
